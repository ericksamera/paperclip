How to capture papers

Paperclip captures pages via the Chrome extension and saves everything locally.

Install the Chrome extension

  1. Open chrome://extensions
  2. Enable Developer mode
  3. Click Load unpacked
  4. Select the folder extensions/chrome from this repo

Capture a paper

  1. Open an article page
  2. Click the Paperclip extension button
  3. Refresh the Library page to see it
